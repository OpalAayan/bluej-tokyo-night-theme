## BlueJ Theme Installation (Quick Guide)

Want to give BlueJ a fresh new look? Follow these quick steps — it only takes a minute!

### 1. Close BlueJ

Make sure BlueJ isn’t running before you start.

### 2. Replace the Old Files

1. Open your BlueJ installation folder.
2. Go into the **lib** folder.
3. Delete or rename the existing **font** folder and the **stylesheet** file (or **bluej.css**).
4. Copy the new **font** folder and **stylesheet** from this theme pack into the same place.
5. If asked, choose **Replace** for any duplicate files.

### 3. Done!

Open BlueJ again — your new theme and fonts should now be active. Enjoy the new look!
